Решенный кейс:

Занятия проводятся в классе на 16 компьютеров. Дом моего прихода каждый из них администрировался вручную. Для того, чтобы обеспечить комфортный опыт обучения для студентов, мной было принято решение объединить компьютеры в домен. Для этого я взял книгу в библиотеке по администрированию Windows Server 2008. Поднял на одном из компьютеров сервер, назначил ему роли ADDS, DNS. Настроил обязательнвый профиль для студентов. Создал групповую политику. Добавил все 16 компьютеров в домен.

Результат:

Проводить занятия стало легче, окружение операционной системы, настройки и набор прикладных программ стали унифицированными. Студенты получили возможность пересаживаться за разные компьютеры, так как ожидают один и тот же опыт использования. Преподавателям стало легче объяснять как использовать программы.
